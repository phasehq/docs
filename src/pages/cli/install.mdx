# Install

Install the Phase CLI on the platform or operating system of your choice.

## Prerequisites

<CodeGroup>

```fish {{ title: 'MacOS' }}
# Install brew - https://brew.sh/
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

```powershell {{ title: 'Windows' }}
# Install scoop - https://scoop.sh
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser # Optional: Needed to run a remote script the first time
irm get.scoop.sh | iex

# Install git - Optional: if you don't have it already
scoop install git
```

```fish {{ title: 'Alpine Linux' }}
apk add --no-cache curl && curl
```

```fish {{ title: 'RedHat/CentOS/Amazon Linux' }}
curl -fsSL https://pkg.phase.dev/install.sh | sh
```

```fish {{ title: 'Ubuntu/Debian' }}
curl -fsSL https://pkg.phase.dev/install.sh | sh
```

```fish {{ title: 'Arch Linux' }}
curl -fsSL https://pkg.phase.dev/install.sh | sh
```

```fish {{ title: 'Pip' }}
# Install python pip: https://pip.pypa.io/en/stable/installation/

# Linux:
python -m ensurepip --upgrade

# MacOS:
python -m ensurepip --upgrade

# Windows:
C:> py -m ensurepip --upgrade
```

```fish {{ title: 'Docker' }}
# Install Docker on your system

# Linux:
curl -fsSL https://get.docker.com | sh

# Others: https://docs.docker.com/engine/install
```

```fish {{ title: 'Raspberry Pi' }}
curl -fsSL https://pkg.phase.dev/install.sh | sh
```

</CodeGroup>

## Installation {{ className: 'lead' }}

<CodeGroup>

```fish {{ title: 'MacOS' }}
brew install phasehq/cli/phase
```

```fish {{ title: 'Windows' }}
scoop bucket add phasehq https://github.com/phasehq/scoop-cli.git
scoop install phase
```

```fish {{ title: 'Alpine Linux' }}
apk add --no-cache curl && curl -fsSL https://pkg.phase.dev/install.sh | sh
```

```fish {{ title: 'RedHat/CentOS/Amazon Linux' }}
curl -fsSL https://pkg.phase.dev/install.sh | sh
```

```fish {{ title: 'Ubuntu/Debian' }}
curl -fsSL https://pkg.phase.dev/install.sh | sh
```

```fish {{ title: 'Arch Linux' }}
curl -fsSL https://pkg.phase.dev/install.sh | sh
```

```fish {{ title: 'Pip' }}
pip3 install phase-cli
```

```fish {{ title: 'Docker' }}
# Run the latest version
docker run phasehq/cli

# Optional: run a specific version - docker run phasehq/cli:<version>
```

```fish {{ title: 'Raspberry Pi' }}
curl -fsSL https://pkg.phase.dev/install.sh | sh
```

</CodeGroup>
